# Proyecto-Integrador-l---Gansos

Este código desarrolla una simulación estocástica basada en teoría de colas para analizar el comportamiento de listas de espera hospitalarias.

## Objetivo
Nuestro objetivo ha sido estudiar cómo influye el número de médicos disponibles en el tiempo de espera de los pacientes mediante un modelo M/M/c implementado en Python.

## Herramientas utilizadas
Las herramientas que hemos usado han sido:
- Python
- Google Colab
- NumPy
- Pandas
- Matplotlib

## Contenido del repositorio
- Notebook principal de la simulación
- Resultados y gráficas obtenidas
- Código fuente del proyecto

Proyecto realizado para la asignatura Proyecto Integrador I – Universidad Francisco de Vitoria.
